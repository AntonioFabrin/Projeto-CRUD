Projeto CRUD com Node.js e Express
Este é um projeto simples de CRUD (Create, Read, Update e Delete) que desenvolvi para praticar conceitos básicos de desenvolvimento web no backend.
A ideia foi entender melhor como funciona a comunicação entre servidor, rotas e interface, além de organizar um projeto de forma mais estruturada.

Tecnologias utilizadas:
Node.js Express.js HTML CSS Bootstrap e JavaScript

O sistema cria novos registros, atualiza, edita e deleta registros

Durante a criação do projeto, aprendi bastante sobre organização de projeto separando rotas, controllers e models.
Uso do Express para lidar com requisições HTTP.
Manipulação de dados entre backend e interface.
Estruturação de um projeto Node.js mais organizada.
Uso de Bootstrap para criar uma interface básica.
